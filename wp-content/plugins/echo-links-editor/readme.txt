Links Editor for PDFs and More

== Description ==

Set Articles to links to PDFs, pages, posts and websites. On KB Main Page, choose icons for your articles.

== Installation ==

= Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: echo-links-editor.zip.
2. Unzip the file echo-links-editor.zip.
3. Upload the 'echo-links-editor' folder to your '/wp-content/plugins' directory (do not rename the folder).

You can also read instructions at https://www.echoknowledgebase.com/documentation/?top-category=add-ons
