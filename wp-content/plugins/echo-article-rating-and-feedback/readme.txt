Let your readers rate the quality of your articles and submit insightful feedback. Utilize analytics on most and least rated articles.

== Description ==

Capture user opinion about the article with a rating system. Also offer feedback form for more detailed user input about article content.

== Installation ==

= Manually Upload Plugin Files =

1. Download the plugin file from the plugin page: echo-article-rating-and-feedback.zip.
2. Unzip the file echo-article-rating-and-feedback.zip.
3. Upload the 'echo-article-rating-and-feedback' folder to your '/wp-content/plugins' directory (do not rename the folder).

You can also read instructions at https://www.echoknowledgebase.com/documentation/?top-category=add-ons
